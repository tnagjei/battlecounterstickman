How to Build Corris Rivett

Step-by-step instructions for building the Corris Rivett from scratch in My Winter Car.

Build Order Overview

  • 1. Chassis: Install subframes and suspension
  • 2. Engine: Mount engine and transmission
  • 3. Fuel System: Connect tank, lines, carburetor
  • 4. Electrical: Wire battery, starter, lights
  • 5. Body: Attach panels, doors, windows
  • 6. Interior: Install seats, dashboard, pedals

Essential Tips

  • Tighten all bolts with the wrench
  • Check oil and coolant before first start
  • Save frequently during the build
  • Work in a warm garage
